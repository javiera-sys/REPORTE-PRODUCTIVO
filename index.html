Guardar en GitHub

Sube tus cambios (datos e imágenes nuevas) al repositorio de GitHub donde vive tu index.html. El HTML/CSS/JS no se vuelven a subir aquí — solo se actualiza data/cambios.json y, si agregaste fotos nuevas, los archivos dentro de data/images/.

Formato usuario/repositorio (no la URL del sitio publicado). Si pegas un link de github.com o de tu página *.github.io, se convierte automáticamente.

Igual que siempre: la ruta de tu index.html (ej. "index.html" o "carpeta/index.html"). A partir de esa ruta, la app guarda los datos en una carpeta "data" junto a él.

Necesita permiso "repo" (o "Contents: Read and write" si es un token fino). Se guarda solo en este navegador, nunca se comparte con nadie más.
